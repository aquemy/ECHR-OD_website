    Multiclass Classification

    The multi-class classification consists in finding a mapping that assigns to each input vector one label among a list of possible labels. The dataset contains instances and output variables, each of them taking value into {0,1}.

    Warning: each output variable indicated that a specific article is violated (1) or not (0).

    Statistic Value

    Multilabel Classification

    The multi-label classification consists in finding a mapping that assigns to each input vector one or several labels. The dataset contains instances and output variables, each of them taking value into {0,1}.

    Warning: each output variable indicated that a specific article is violated (1) or not (0). If a variable does not appear in the list for a specific case it means the article was not discussed.

    Statistic Value

    Dataset structure

    Descriptive variables

    The descriptive variables are describing the cases. They are provided by HUDOC directly for most, or extracted from the judgement document for some.

    Name ID Description

    Outcomes variables

    The outcomes variables are binary variables indicating if there is a violation (value 1) or no violation (value 0) of an article. If a variable is absent from a case, it means the case was not about this article. The variable with ID 0 correspond to the case ID for reference purpose.

    Article ID Description

    Example: 
    0:8707 12:1 3:0 4:0
    This indicated that the the judgement of case ID 8707 states a violation of the article with ID 12 and no violation for the articles with ID 3 and 4.
